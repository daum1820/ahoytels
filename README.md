
### Ahoytels
######Fake Hotel - Javascript Project

#### Environment
* NodeJS
* Yarn

#### IDE
* Visual Studio Code

#### Client Side
* Javascript
* Axios
* Moment
* Lodash
* Webpack
* Babel

#### Commands
###### Install
```shell
$ yarn
```
###### Development
```Test
$ yarn dev
```
###### Build
```shell
$ yarn build
```
###### Production
```shell
$ yarn start
```
